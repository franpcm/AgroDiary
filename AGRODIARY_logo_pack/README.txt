AGRODIARY Logo Pack
===================

Contenido:
- png/ : PNG con fondo transparente (varios anchos).
- jpg/ : JPG con fondo blanco/crema (útil para Office/impresión rápida).
- pdf/ : PDF a 300 dpi (logo sobre fondo blanco/crema).
- svg/ : SVG con el PNG embebido (NO es vector real; sirve para compatibilidad).
- favicon/ : favicon.ico + PNGs (16/32/48/64) + iconos para PWA (192/512) usando el isotipo (hoja).
- social/ : imágenes para OpenGraph / redes.

Notas:
- Como el archivo de origen es un PNG raster, los tamaños grandes (por encima del original) están reescalados.
- Si necesitas un SVG/PDF vectorial “real” (para imprenta), lo ideal es disponer del archivo original en vector (AI/SVG/PDF).
